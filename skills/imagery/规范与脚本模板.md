## 亚马逊图文视频规范与创意模板

### 主图规范 checklist
- 纯白背景（RGB 255/255/255），无灰底/彩底/场景。
- 产品占画面 ≥ 85%，居中，完整不出血。
- **禁止**任何文字、水印、logo 角标、促销贴纸、边框。
- 长边 ≥ 1600px（支持悬停放大），JPEG/PNG/TIFF。
- 只展示实际售卖的产品，不含赠品未标注、不含他人商标。

### 场景图 / 副图规范
- 可展示使用场景、细节、尺寸对比、卖点标注，但不得虚假宣传。
- 文字标注需真实、可验证；不得出现极限词与医疗夸大。

### 绘图 prompt 结构（中英各一版）
`主体 + 材质/颜色 + 关键细节 + 光影 + 背景 + 镜头/构图 + 风格/质感 + 分辨率`
- 主图版：`... on pure white seamless background, studio softbox lighting, centered, high detail, 4k product photography`
- 场景图版：加入真实使用场景、人物手部/环境，但主体清晰。

### 构图配色拆解要点
- 构图：主体位置（三分法/居中）、留白比例、视觉引导线、层次。
- 配色：主色 / 辅助色 / 点缀色，冷暖对比，与类目调性是否匹配。
- 输出「现状 → 问题 → 优化建议」三段。

### 短视频脚本模板
**开箱**（15–30s）：包装外观 → 开箱瞬间 → 产品全貌 → 核心卖点特写 → 使用一瞬 → 收尾 CTA。
**测评**（30–60s）：痛点引入 → 产品登场 → 关键功能实测 → 对比/数据 → 使用体验 → 结论 CTA。
