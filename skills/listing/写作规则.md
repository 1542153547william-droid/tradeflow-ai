## Listing 写作规则

### 标题公式
`品牌 + 核心关键词 + 关键属性/材质 + 适用场景/机型 + 主要卖点`
- 长度控制在站点上限内（US ≤ 200 字符），核心词前置。
- 不堆砌关键词、不使用极限词（best/no.1…）、不出现他人品牌。

### 五点描述（Bullet Points）结构
每条一个主题，**痛点 → 卖点 → 利益**：
1. 核心功能 / 最大卖点
2. 材质 / 工艺 / 耐用性
3. 兼容性 / 适用范围
4. 使用场景 / 体验提升
5. 售后 / 保障 / 品牌承诺（合规，不写"guaranteed"绝对承诺）

### 三套写作逻辑（2.7）
| 品况 | 侧重 | 语气 |
|---|---|---|
| **新品** | 卖点教育、建立信任、突出创新点 | 有说服力、讲清"为什么选它" |
| **成熟品** | 差异化、评价背书、关键词广覆盖抢流量 | 稳、强调口碑与销量 |
| **清仓** | 性价比、限时、库存紧迫感（不虚假降价） | 紧凑、行动导向 |

### 关键词
- 先用 `inject_keywords` 看该类目 candidates，把 missing 的高优先词根埋进标题+五点。
- 覆盖到位但自然：宁可少埋、读得顺，也不硬塞。

### 合规红线
- 生成后必须过 `compliance_gate`；命中就用其 `合规替代` 改写，最多 3 次。
- 医疗夸大、他人商标/IP、极限词一律不出现。

### 范例（手机壳·新品·US，节选）
- Title: `BestNest Shockproof Clear Case for iPhone 15 - Slim Protective Cover with Raised Edges`
- Bullet 1: `MILITARY-GRADE PROTECTION – shockproof corners absorb drops so your phone stays safe...`
